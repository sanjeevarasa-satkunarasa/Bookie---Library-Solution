Help and Documentation

Registering a book

  1. Fill out the form
  2. Press the submit button
  3. The list of the library's books should now be updated!

Borrowing a book

  1. Select the book you want to borrow from the list
  2. Press the "Borrow Book" button
  3. The list of the library's books should now be updated!

Returning a book

  1. Select the book you want to return from the list
  2. Press the "Return Book" button
  3. The list of the library's books should now be updated!
